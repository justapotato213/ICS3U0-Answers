1. Network card recieves information about the shipping label.
  2. The network card decodes information into information useful to the PC.
  3. Network card sends data to the motherboard South Bridge via PCIe.
  4. Southbridge sends an interupt signal to the CPU and sends information to the North Bridge, then to the CPU's cache. (Very fast memory, but very small in size, located within the CPU)
  5. The ALU processes the information, being able to use logic gates and arithmetic to create a certain output. (It processes things every clock tick, and clock rate determines how fast the clock tick is.)
  6. The ALU sends the output to the Memory Management Unit.
  7. The MMU sends the output into RAM from cache, translating the virtual memory address sent from the CPU, and sends it to a physical space in RAM.
  8. The CPU then tells the RAM to move it to non-volatile storage.
  9. The RAM communicates with the North bridge to move information to the non-volatile storage.
  10. The North Bridge sends the information to the South Bridge.
  11. The South Bridge sends the information over the SATA cable to the storage.
  12. The storage reads the information, and its own controller tells it to store it in a certain area.
  13. The OS tells the CPU to print the new information via an interrupt signal.
  14. The MMU tells the RAM to move the information from the disk into RAM.
  15. The RAM communicates with the North Bridge.
  16. The North Bridge communicates with the South Bridge.
  17. The South Bridge communicates with the storage to find the information.
  18. The storage controller finds the information, and sends it to the South Bridge.
  19. The South Bridge sends the information to the North Bridge.
  20. The North Bridge sends the information into RAM.
  21. The MMU moves the information from the RAM into the cache.
  22. The CPU with the ALU specifically combines this information as well as the information about the printer to send to the network card.
  23. The CPU sends information to the South Bridge on the motherboard.
  24. Southbridge processes the information to send to the network card.
  25. The Southbridge sends the information over PCIe to the network card.
  26. The network card recieves the information and decodes it.
  27. The network cards finds the address it is looking for on the network.
  28. The network card encodes the information, and sends it as packets over the network to the printer.
